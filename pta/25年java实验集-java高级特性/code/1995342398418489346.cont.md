# 容器-HashMap-排序
输入多个学生的成绩信息，包括：学号、姓名、成绩。

学号是每个学生的唯一识别号，互不相同。

姓名可能会存在重复。

要求：使用HashMap存储学生信息。<br>

### 输入格式:

输入多个学生的成绩信息，每个学生的成绩信息格式：学号+英文空格+姓名+英文空格+成绩

以“end”为输入结束标志

### 输出格式:

按学号从大到小的顺序输出所有学生信息，每个学生信息的输出格式：学号+英文空格+姓名+英文空格+成绩

### 输入样例:

在这里给出一组输入。例如：

```in
20201124 张少军 83
20201136 李四 78
20201118 郑觉先 80
end
```

### 输出样例:

在这里给出相应的输出。例如：

```out
20201136 李四 78
20201124 张少军 83
20201118 郑觉先 80
```
