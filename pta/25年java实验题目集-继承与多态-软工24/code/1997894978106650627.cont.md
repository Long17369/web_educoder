# 使用接口完成银行取钱。
利用面向接口编程思想实现一个银行取钱程序。
（1）定义一个接口银行IBank，包含抽象方法取钱double withdraw(double money);定义两个实现类：工商银行ICBC、农业银行ABC，钱在银行的利息不同，工商银行利息是千分之一，农业银行利息是千分之二。
（2）定义一个类客户Customer，包含方法从银行取钱double getMoneyFromBank(IBank bank,double money)，通过接口回调从不同银行取钱。
（3）编写main方法，根据输入的银行和取款数，输出从不同银行取钱时实际取得的金额。

### 输入格式:

给出银行名字和取款数。

### 输出格式:

取出的实际金额。

### 输入样例:

在这里给出一组输入。例如：

```in
ABC 1000
```

### 输出样例:

在这里给出相应的输出。例如：

```out
1002.0
```
### 输入样例:

在这里给出一组输入。例如：

```in
ICBC 1000
```

### 输出样例:

在这里给出相应的输出。例如：

```out
1001.0
```
