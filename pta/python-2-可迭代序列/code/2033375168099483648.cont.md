# 《西游记》人物的粉丝数
《西游记》是中国古代第一部浪漫主义章回体长篇神话小说，达到了古代长篇浪漫主义小说的巅峰，是我国的四大名著之一，流传极广，脍炙人口。书中众多神魔，都有各自的粉丝。从键盘上读入一组粉丝喜好西游记角色的名字（以半角逗号分割），统计出对应的粉丝数。

### 输入格式:

一组粉丝喜好西游记角色的名字（以半角逗号分割），读入人物名字左侧或右侧可能含有一个或多个空格(用方法 字符串.strip() 去除字符串头尾的空格)。

### 输出格式:

西游记角色名字:粉丝数 #所有符号均为半角符号，按照粉丝数从高到低排列。

### 输入样例:

在这里给出一组输入。例如：

```in
唐僧 ,孙悟空,   猪八戒,  财神,沙僧, 孙悟空, 猪八戒, 孙悟空 ,白骨精,  猪八戒 ,财神,二郎神,孙悟空,二郎神,猪八戒  ,孙悟空  ,财神,  孙悟空
```

### 输出样例:

在这里给出相应的输出。例如：

```out
孙悟空:6
猪八戒:4
财神:3
二郎神:2
白骨精:1
沙僧:1
唐僧:1
```
