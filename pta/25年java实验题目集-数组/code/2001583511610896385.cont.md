# 不等长二维数组
二维数组有多个一维数组构成（不一定长度相等）。
求出二维数组的最小值min。
将二维数组的每个元素的值修改为原来的值减去min。并输出整个二维数组。 

### 输入格式:

先输入一个整数row，表示二维数组有多少个一维数组构成。
然后输入row行。每行先输入一个整数column，表示一维数组的长度。然后输入column个整数，为此一维数组的元素。
由上面row个一维数组，构成一个二维数组。 

### 输出格式:

求出二维数组的最小值min。
将二维数组的每个元素的值修改为原来的值减去min。
输出整个二维数组，构成二维数组的每个一维数组单独一行，每两个元素用空格分开。 

### 输入样例:

```in
3
4 2 3 4 5
3 3 2 1
2 6 7
```

### 输出样例:

```out
1 2 3 4
2 1 0
5 6
```
